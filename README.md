# Internships Page — Redesign

A redesigned, more visually engaging version of the VaultofCodes internship listing page, built as a VaultofCodes internship task. Recreates the original card layout with a fresh, more attractive look using only HTML and CSS.

🔗 **Live demo:** https://utkashpawar.github.io/Internship-Page/

## About

The original page listed four internship tracks (Web Development, App Development, Python Programming, Java Programming) in a plain card grid. This redesign keeps the same information but reworks the visual design to feel more polished and "stunning," per the task brief — while sticking to HTML + CSS only, with no header, footer, or navbar (per the assignment's scope).

## Internship Tracks Shown

- **Web Development Internship Program** — 1–2 months, Virtual, starts 05/11/2023
- **App Development Internship Program** — 1–2 months, Virtual, starts 05/11/2023
- **Python Programming Internship Program** — 1–2 months, Virtual, starts 05/11/2023
- **Java Programming Internship Program** — 1–2 months, Virtual, starts 05/11/2023

Each card includes a working **View Details** button/link.

## Features

- 🎨 Custom card design — reworked visual treatment of the original plain layout
- 🔘 Working "View Details" buttons/links on every card
- 📐 Clean use of the CSS box model for consistent card spacing and structure
- 🚫 No header, footer, or navbar — matches the assignment's scope exactly (webpage content only)
